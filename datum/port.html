/tg/ Station 13 - Modules - Types — Var Details - Proc Details

Component Port
/datum/port

A base type port used by a component

Connects to other ports. This is an abstract type that should not be instanciated

Vars

colorThe port color. If unset, appears as blue.
connected_componentThe component this port is attached to
datatypeThe port type. Ports can only connect to each other if the type matches
default_datatypeThe default port type. Stores the original datatype of the port set on Initialize.
nameName of the port. Used when displaying the port.

Procs

convert_valueReturns the value to be set for the port
datatype_to_colorConverts the datatype into an appropriate colour
disconnectDisconnects a port from all other ports
set_datatypeSets the datatype of the port.
