proc
check_compatibility
()

Checks compatibility of implant against the cyberlink

Every implant contains it's own encode_info, this info stores the data on what security, encoding and operating protocols it uses. Implant is compatible if for every protocol catergory it shares at least 1 protocol in common with the link. If it fails to meet that criteria, than it is incompatible and this proc returns FALSE. If it is compatibile returns TRUE
