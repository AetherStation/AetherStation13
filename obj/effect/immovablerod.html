Proc Details

proc
go_for_a_walk
(walkies_location)

Allows your rod to release restraint level zero and go for a walk.

If walkies_location is set, rod will walk_towards the location, chasing it across z-levels if necessary. If walkies_location is not set, rod will call complete_trajectory() and follow the logic from that proc.

Arguments:

  • walkies_location - Any atom that the immovable rod will now chase down as a special target.
